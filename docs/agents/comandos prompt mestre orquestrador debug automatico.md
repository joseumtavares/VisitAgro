Modo de uso para qualquer pessoa

A pessoa só precisa escrever algo assim:

Execute o fluxo automatizado de debugging com comentários técnicos para o módulo de representantes.

Ou:

Execute o fluxo automatizado de debugging com comentários técnicos para todo o sistema, em blocos.
Exemplo de uso real
Entrada
Quero automatizar o debug do módulo de vendas e deixar o código mais fácil de investigar.
O sistema faz sozinho
cria lote;
detecta arquivos de vendas;
comenta pontos críticos;
revisa comentários;
remove ruído;
consolida;
valida;
fecha o lote.
Resultado prático

Você passa a ter um pipeline onde:

qualquer pessoa consegue iniciar;
o sistema decide a ordem;
os agentes fazem o trabalho técnico;
a saída fica documentada;
o risco de ruído cai;
o código fica melhor para investigação futura.
