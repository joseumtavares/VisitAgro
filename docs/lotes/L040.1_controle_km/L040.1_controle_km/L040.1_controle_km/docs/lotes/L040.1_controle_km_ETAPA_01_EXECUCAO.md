## 3. Causa Raiz

A tabela `km_logs` existia no schema de referência mas faltavam: `workspace` (impedindo tenancy correta), `created_at`/`updated_at`, índice único diário e constraint de integridade. Não existiam rota de API nem página de dashboard. A migration é incremental e idempotente.
